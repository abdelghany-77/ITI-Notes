Part I: Operating Systems Fundamentals
Q1
The Main Memory consists of RAM (Random Access Memory) and ROM (Read Only Memory).
✓ TRUE
FALSE

Correct Answer: TRUE

Main memory indeed consists of both RAM (volatile, temporary storage) and ROM (non-volatile, permanent storage containing basic system instructions).

Q2
All of the secondary storages used as Output devices only for permanent storage.
TRUE
✓ FALSE

Correct Answer: FALSE

Secondary storage devices serve as both input and output devices, not just output devices.

Q3
Bootstrap is a framework stored in ROM contains some instructions called basic input output instructions (BIOS).
✓ TRUE
FALSE

Correct Answer: TRUE

Bootstrap program is stored in ROM and contains BIOS instructions for basic system initialization.

Q4
Windows server 2008 and Novell Netware are batching operating systems.
TRUE
✓ FALSE

Correct Answer: FALSE

These are network operating systems, not batch operating systems.

Q5
The purpose of the operating system is to ensure that it abstracts the HW and facilitates the seamless execution of users' applications using the system.
✓ TRUE
FALSE

Correct Answer: TRUE

This is the fundamental purpose of an operating system - to abstract hardware and manage applications.

Q6
The application itself is the responsible for accessing the memory variable storage and perform the calculations based on values in memory.
✓ TRUE
FALSE

Correct Answer: TRUE

Applications are responsible for their memory management and computational operations.

Q7
Threads help in performing parallelism within the same process.
✓ TRUE
FALSE

Correct Answer: TRUE

Threads enable concurrent execution within a single process, achieving parallelism.

Q8
Interrupt vector contains the addresses for all Interrupt Service Routines (ISR) for all I/O devices that connected to the computer system.
✓ TRUE
FALSE

Correct Answer: TRUE

The interrupt vector table stores addresses of ISRs for handling different interrupts from I/O devices.

Q9
The process which running in user-mode services would be able to access the hardware directly.
TRUE
✓ FALSE

Correct Answer: FALSE

User-mode processes cannot access hardware directly; they must use system calls through the OS.

Q10
A Virtual Machine has its own guest OS, which may be the same as or different from the underlying host OS.
✓ TRUE
FALSE

Correct Answer: TRUE

Virtual machines can run any compatible operating system, independent of the host OS.

Q11
The kernel of an operating system always runs in privileged mode to access system resources.
✓ TRUE
FALSE

Correct Answer: TRUE

The kernel operates in privileged mode (kernel mode) to have unrestricted access to hardware and system resources.

Q12
Context switching is the process of storing and restoring the state of a process when switching between processes.
✓ TRUE
FALSE

Correct Answer: TRUE

Context switching involves saving the current state of a process and loading the state of another process to enable multitasking.

Q13
Paging and segmentation are both memory management techniques that can eliminate external fragmentation.
TRUE
✓ FALSE

Correct Answer: FALSE

Paging eliminates external fragmentation, but segmentation can still cause external fragmentation.

Q14
The critical section problem occurs when multiple processes access shared data simultaneously.
✓ TRUE
FALSE

Correct Answer: TRUE

Critical section refers to code segments where shared resources are accessed, requiring synchronization to prevent race conditions.

Q15
Device drivers are always part of the operating system kernel and cannot be loaded dynamically.
TRUE
✓ FALSE

Correct Answer: FALSE

Modern operating systems support dynamic loading of device drivers, and they can be loaded as kernel modules or run in user space.

Q16
Which of the following are Output Devices? (Select all correct answers)
a. Data Projectors
b. Microphone
c. Speaker and Headphones
d. Scanner (2D and 3D)
e. Printer (2D and 3D) - inkjet or laser

Correct Answers: a, c, e

Data projectors, speakers/headphones, and printers are output devices. Microphones and scanners are input devices.

Q17
The __________ is a program that controls the execution of other programs running on the system. It acts as a facilitator and intermediate layer between the different software components and the computer hardware.
a. Bootstrap
b. Operating System
c. Database Management System
d. Embedded System

Correct Answer: b. Operating System

The operating system manages and controls the execution of all programs and acts as an intermediary between software and hardware.

Q18
The main objectives of any operating systems are: (Select all correct answers)
a. Efficiency of the OS in terms of responsiveness, fluidity, and so on
b. Ability to abstract and extend to new devices and software
c. Ease of usability to the user in terms of making it convenient
d. Control the flow of data through local networks

Correct Answers: a, b, c

Main OS objectives include efficiency, abstraction capability, and usability. Network data flow control is not a primary OS objective.

Q19
The Usage type categorization of operating systems are: (Select all correct answers)
a. Batch Operating Systems
b. Time Sharing Operating Systems
c. Parallel-Distributed Operating Systems
d. Network Operating Systems
e. Real Time Operating Systems

Correct Answers: a, b, c, d, e

All listed options are valid categorizations of operating systems based on their usage types.

Q20
Select the Advantages of virtualization from the following: (Select all correct answers)
a. Emulate more machines than are physically available
b. Run operating systems where the physical hardware is unavailable
c. Run legacy systems
d. Debug problems (suspend and resume the problem machine)
e. All of the above

Correct Answer: e. All of the above

All listed options are advantages of virtualization technology.

Q21
Which of the following are the responsibilities of an operating systems? (Select all correct answers)
a. Access and protection
b. File systems
c. Scheduling
d. Memory management
e. I/O and resource management

Correct Answers: a, b, c, d, e

All listed options are core responsibilities of an operating system.

Q22
The types of addressing of memory in the operating systems are: (Select all correct answers)
a. Logical Address
b. Physical Address
c. Controlling
d. None of the above

Correct Answers: a, b

Operating systems use logical addresses (virtual) and physical addresses (actual memory locations).

Q23
The way of management when the two processes need to save in the shared memory at the same time, it is called______.
a. Compilation
b. Execution
c. Synchronization
d. Termination

Correct Answer: c. Synchronization

Synchronization manages concurrent access to shared resources between processes.

Q24
When a set of processes become blocked because each process is holding a resource and waiting for another resource acquired by some other process. This is called ______.
a. Synchronization
b. Deadlock
c. Acceleration
d. Process Status

Correct Answer: b. Deadlock

Deadlock occurs when processes are permanently blocked, each waiting for resources held by others.

Q25
The OS needs to be able to periodically clear temporarily created files that may no longer be required for the functioning of the system. This is typically done using a ______ on the system.
a. Garbage Collector
b. Data Collector
c. Anti-Virus
d. None of the above

Correct Answer: a. Garbage Collector

Garbage collectors automatically manage memory and clean up unused resources in the system.

Part II: Computer Networks Fundamentals and Security
Q26
We can find our IP address on a MS Windows based computer, by using:
a. Control Panel\Network and Internet\Network and Sharing Center
b. Ipconfig Command
c. Ipconfig /all Command
d. All the above

Correct Answer: d. All the above

All methods can be used to find IP address on Windows systems.

Q27
We are using__________ protocol to read our mails when internet connection is NOT available all the time.
a. POP3
b. IMAP4
c. UDP
d. FTP

Correct Answer: a. POP3

POP3 downloads emails to local device, allowing offline reading. IMAP requires constant connection.

Q28
What is the name of the attack method that can be used to crack passwords?
a. DOS attack
b. Brute force attack
c. Smurf attack
d. Spam

Correct Answer: b. Brute force attack

Brute force attacks systematically try all possible password combinations to crack passwords.

Q29
To communicate with hosts on the same LAN, we must use a__________.
a. DNS
b. Router
c. Switch
d. DHCP

Correct Answer: c. Switch

Switches connect devices within the same LAN (Local Area Network).

Q30
You must have an IP address to connect to the internet, you can get your IP address (if your configuration was "obtain automatically") from:
a. Router
b. DHCP
c. DNS
d. Switch

Correct Answer: b. DHCP

DHCP (Dynamic Host Configuration Protocol) automatically assigns IP addresses to devices.

Q31
What is the benefit of the Networking?
a. File Sharing
b. Easier access to Resources
c. Easier Backups
d. All of the Above

Correct Answer: d. All of the Above

Networking provides all mentioned benefits: file sharing, resource access, and easier backups.

Q32
Which of the following is not the Networking Devices?
a. Switch
b. Linux
c. Routers
d. Firewalls

Correct Answer: b. Linux

Linux is an operating system, not a networking device. Others are networking hardware/software.

Q33
A collection of computers and devices connected together via communication devices and transmission media is called a:
a. Workgroup
b. Server
c. Mainframe
d. Network

Correct Answer: d. Network

A network is defined as a collection of interconnected computers and devices.

Q34
Which of the following devices is dedicated to connect several computers wirelessly?
a. Hub
b. Switch
c. Access Point
d. Router

Correct Answer: c. Access Point

Access Points (APs) are specifically designed to provide wireless connectivity to multiple devices.

Q35
An intrusion detection system (IDS) is primarily designed to perform what function?
a. prevents system failures
b. Rate system performance
c. Test a system for vulnerabilities
d. Detect abnormal activity

Correct Answer: d. Detect abnormal activity

IDS systems monitor network traffic and system activities to detect suspicious or malicious behavior.

Q36
__________ is ensuring that information is not revealed to unauthorized persons.
a. Confidentiality
b. Integrity
c. Security
d. Authorization

Correct Answer: a. Confidentiality

Confidentiality is one of the CIA triad principles, ensuring information remains private and secure.

Q37
Which of The Following Malicious Program is Replicate Automatically?
a. Antivirus
b. backdoor
c. Worm
d. Zombie

Correct Answer: c. Worm

Worms are malicious programs that can replicate and spread automatically across networks without human intervention.

Q38
The modern TCP/IP Protocol Suite model consists of __________ Layers?
a. 4 layers
b. 5 layers
c. 6 layers
d. 7 layers

Correct Answer: a. 4 layers

The TCP/IP model has 4 layers: Application, Transport, Internet, and Network Access (Link).

Q39
In which class the IP address 70.70.70.70 exists?
a. Class A
b. Class B
c. Class C
d. Class D

Correct Answer: a. Class A

IP addresses 1-126 in the first octet belong to Class A. 70.x.x.x falls in Class A range.

Q40
If one site fails in distributed system then ______________.
a. the remaining sites can continue operating
b. all the sites will stop working
c. directly connected sites will stop working
d. none of the mentioned

Correct Answer: a. the remaining sites can continue operating

This is a key advantage of distributed systems - fault tolerance and continued operation despite individual site failures.

Q41
Which protocol is used for secure file transfer over the internet?
a. FTP
b. HTTP
c. SFTP
d. Telnet

Correct Answer: c. SFTP

SFTP (SSH File Transfer Protocol) provides secure file transfer with encryption, unlike regular FTP.

Q42
What is the standard port number for HTTPS?
a. 80
b. 21
c. 443
d. 25

Correct Answer: c. 443

HTTPS uses port 443 for secure HTTP communications. HTTP uses port 80.

Q43
Which of the following is a symmetric encryption algorithm?
a. AES
b. RSA
c. DSA
d. ECC

Correct Answer: a. AES

AES (Advanced Encryption Standard) is a symmetric encryption algorithm. RSA, DSA, and ECC are asymmetric.

Q44
What does MAC stand for in network security?
a. Media Access Control
b. Message Authentication Code
c. Machine Access Code
d. Multiple Access Channel

Correct Answer: b. Message Authentication Code

In security context, MAC refers to Message Authentication Code used to verify message integrity and authenticity.

Q45
Which layer of the OSI model is responsible for end-to-end communication?
a. Network Layer
b. Transport Layer
c. Session Layer
d. Application Layer

Correct Answer: b. Transport Layer

The Transport Layer (Layer 4) provides end-to-end communication services and handles data segmentation and reassembly.

Part III: Software Testing Fundamentals
Q46
Which testing phase tests individual software modules combined together as a group?
a. Module testing
b. Integration testing
c. White Box testing
d. Software testing

Correct Answer: b. Integration testing

Integration testing focuses on testing the interfaces and interaction between integrated modules.

Q47
The main focus of acceptance testing is:
a. Finding faults in the system
b. Ensuring that the system is acceptable to all users
c. Testing the system with other systems
d. Testing for a business perspective
e. Testing by an independent test team

Correct Answer: d. Testing for a business perspective

Acceptance testing validates that the system meets business requirements and is ready for deployment.

Q48
Which is NOT true - The black box tester:
a. Should be able to understand a functional specification or requirements document
b. Should be able to understand the source code
c. Is highly motivated to find faults
d. is creative to find the system's weaknesses

Correct Answer: b. Should be able to understand the source code

Black box testing doesn't require knowledge of internal code structure; it focuses on functionality.

Q49
Beta testing is:
a. Performed by customers at their own site
b. Performed by customers at their software developer's site
c. Performed by an independent test team
d. Useful to test bespoke software
e. Performed as early as possible in the lifecycle

Correct Answer: a. Performed by customers at their own site

Beta testing is conducted by actual users in their real environment before final release.

Q50
Verification is:
a. Checking that we are building the right system
b. Checking that we are building the system right
c. Performed by an independent test team
d. Making sure that it is what the user really wants

Correct Answer: b. Checking that we are building the system right

Verification ensures the system is built according to specifications. Validation checks if it's the right system.

Q51
The difference between re-testing and regression testing is:
a. Re-testing is running a test again regression testing looks for unexpected side effects
b. Re-testing looks for unexpected side effects: regression testing is repeating those tests
c. Re-testing is done after faults are fixed regression testing is done earlier
d. Re-testing uses different environments, regression testing uses the same environment
e. Re-testing is done by developers, regression testing is done by independent testers

Correct Answer: a. Re-testing is running a test again regression testing looks for unexpected side effects

Re-testing verifies fixes work; regression testing ensures changes don't break existing functionality.

Q52
Before launching a software which testing is to be done in-house?
a. Beta
b. Gamma
c. Alpha
d. None of the above

Correct Answer: c. Alpha

Alpha testing is performed in-house by internal teams before external beta testing.

Q53
Software testing activities should start
a. As soon as the code is written
b. During the design stage
c. When the requirements have been formally documented
d. As soon as possible in the development life cycle

Correct Answer: d. As soon as possible in the development life cycle

Early testing reduces costs and improves quality. Testing should begin with requirements analysis.

Q54
Faults found by users are due to:
a. Poor quality software
b. Poor software and poor testing
c. Bad luck
d. Insufficient time for testing

Correct Answer: b. Poor software and poor testing

User-found faults typically result from both inadequate software quality and insufficient testing coverage.

Q55
In which order should tests be run?
a. The most important tests first
b. The most difficult tests first (to allow maximum time for fixing)
c. The easiest tests first (to give initial confidence)
d. The order they are thought of

Correct Answer: a. The most important tests first

Testing should be prioritized by risk and importance to ensure critical functionality is verified first.

Q56
Which testing technique is based on the internal structure of the software?
a. Black Box Testing
b. White Box Testing
c. Gray Box Testing
d. Acceptance Testing

Correct Answer: b. White Box Testing

White box testing examines the internal structure, code paths, and logic of the software.

Q57
What is the primary purpose of smoke testing?
a. To ensure basic functionality works before detailed testing
b. To test performance under load
c. To test security vulnerabilities
d. To test user interface design

Correct Answer: a. To ensure basic functionality works before detailed testing

Smoke testing is a preliminary test to check if the basic functionality works before proceeding with comprehensive testing.

Q58
Which of the following is NOT a software testing level?
a. Unit Testing
b. Integration Testing
c. System Testing
d. Compilation Testing

Correct Answer: d. Compilation Testing

Compilation testing is not a recognized testing level. The main levels are Unit, Integration, System, and Acceptance testing.

Q59
What does UAT stand for in software testing?
a. Unit Acceptance Testing
b. User Acceptance Testing
c. Universal Access Testing
d. Unified Application Testing

Correct Answer: b. User Acceptance Testing

UAT is User Acceptance Testing, where end users test the system to ensure it meets their requirements.

Q60
Which testing approach involves testing without any knowledge of the internal workings?
a. Black Box Testing
b. White Box Testing
c. Gray Box Testing
d. Clear Box Testing

Correct Answer: a. Black Box Testing

Black box testing focuses on input-output behavior without knowledge of internal code structure or implementation details.
