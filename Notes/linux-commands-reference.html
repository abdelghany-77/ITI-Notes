File and Directory Operations

uname
Prints system information
$ uname
Linux
$ uname -a
Linux ubuntu 5.4.0-80-generic #90-Ubuntu SMP Fri Jul 9 22:49:44 UTC 2021 x86_64 x86_64 x86_64 GNU/Linux
uname -a prints all system information including kernel name, hostname, kernel release, version, machine hardware name, and operating system.

whoami
Prints the effective user ID
$ whoami
abdelghany-77
pwd
Prints the working directory's full path
$ pwd
/home/abdelghany-77/Documents
cd
Changes the current directory
$ cd /home/abdelghany-77
$ cd ~ # Go to home directory
$ cd .. # Go up one level
$ cd - # Go to previous directory
cd without arguments takes you to your home directory. Use ~ as shorthand for home directory.
ls
Lists directory contents with various options
$ ls
Documents Downloads Music Pictures Videos
$ ls -la
total 28
drwxr-xr-x 7 abdelghany-77 abdelghany-77 4096 Aug 4 21:40 .
drwxr-xr-x 3 root root 4096 Aug 1 10:00 ..
-rw-r--r-- 1 abdelghany-77 abdelghany-77 220 Aug 1 10:00 .bash_logout
drwxr-xr-x 2 abdelghany-77 abdelghany-77 4096 Aug 4 15:30 Documents
$ ls -lh # Human readable sizes
$ ls -R # Recursive listing
$ ls -i # Show inode numbers
cat
Opens and displays files on the same page
$ cat /etc/hostname
ubuntu-server
$ cat -n /etc/passwd | head -5
1 root:x:0:0:root:/root:/bin/bash
2 daemon:x:1:1:daemon:/usr/sbin:/usr/sbin/nologin
3 bin:x:2:2:bin:/bin:/usr/sbin/nologin
4 sys:x:3:3:sys:/dev:/usr/sbin/nologin
5 sync:x:4:65534:sync:/bin:/bin/sync
head / tail
Shows first/last lines of files
$ head -5 /etc/passwd
root:x:0:0:root:/root:/bin/bash
daemon:x:1:1:daemon:/usr/sbin:/usr/sbin/nologin
bin:x:2:2:bin:/bin:/usr/sbin/nologin
$ tail -3 /var/log/syslog
$ tail -f /var/log/syslog # Follow log in real-time
cp
Copies files and directories
$ cp file1.txt file2.txt
$ cp -r /home/user/docs /backup/
$ cp -i important.txt backup.txt # Interactive mode
Use -r for recursive copying of directories, -i for interactive confirmation.
mv
Moves and renames files and directories
$ mv oldname.txt newname.txt
$ mv file1.txt /home/user/Documents/
$ mv -i source.txt destination.txt
touch
Creates files or updates timestamps
$ touch newfile.txt
$ touch file1.txt file2.txt file3.txt
$ touch report_{1..5}.txt # Creates report_1.txt through report_5.txt
mkdir
Creates directories
$ mkdir new_directory
$ mkdir dir1 dir2 dir3
$ mkdir -p projects/web/css # Creates parent directories
rm / rmdir
Removes files and directories
$ rm file.txt
$ rm -i important.txt # Interactive removal
$ rm -r old_directory # Recursive removal
$ rmdir empty_directory # Removes empty directories only
Be very careful with rm -r as it permanently deletes files and directories!
ln
Creates hard and symbolic links
$ ln file.txt hardlink.txt # Hard link
$ ln -s /path/to/file symlink.txt # Symbolic link
$ ls -l symlink.txt
lrwxrwxrwx 1 user user 13 Aug 4 21:40 symlink.txt -> /path/to/file

User & Group Management

useradd
Adds new users to the system
$ sudo useradd john
$ sudo useradd -m -s /bin/bash -c "John Doe" john
$ useradd -D # View default settings
Use -m to create home directory, -s to set shell, -c for comment/full name.
passwd
Changes or sets user passwords
$ passwd # Change your own password
$ sudo passwd john # Change john's password
New password:
Retype new password:
passwd: password updated successfully
usermod
Modifies user account properties
$ sudo usermod -aG sudo john # Add john to sudo group
$ sudo usermod -L john # Lock john's account
$ sudo usermod -U john # Unlock john's account
groupadd
Creates new groups
$ sudo groupadd developers
$ sudo groupadd -r system_group # System group
chown
Changes file/directory ownership
$ sudo chown john file.txt
$ sudo chown john:developers project.txt
$ sudo chown -R john:john /home/john/
chmod
Changes file/directory permissions
$ chmod 755 script.sh
$ chmod +x program
$ chmod u+w,g-w,o-r file.txt
$ chmod -R 644 /var/www/html/
755 = rwxr-xr-x, 644 = rw-r--r--. Use +/- to add/remove permissions.
su
Switch user accounts
$ su john # Switch to john (keep environment)
$ su - john # Switch to john (john's environment)
$ su - # Switch to root
id
Displays user and group IDs
$ id
uid=1000(abdelghany-77) gid=1000(abdelghany-77) groups=1000(abdelghany-77),4(adm),24(cdrom),27(sudo)
$ id john

System & Process Management

ps
Lists running processes
$ ps
PID TTY TIME CMD
1234 pts/0 00:00:00 bash
5678 pts/0 00:00:00 ps
$ ps aux # All processes for all users
$ ps -ef # All processes with parent PIDs
top
Displays dynamic process information
$ top
top - 21:40:35 up 1:23, 2 users, load average: 0.01, 0.02, 0.05
Tasks: 95 total, 1 running, 94 sleeping, 0 stopped, 0 zombie
%Cpu(s): 0.3 us, 0.7 sy, 0.0 ni, 99.0 id, 0.0 wa, 0.0 hi, 0.0 si, 0.0 st
Press 'q' to quit top, 'k' to kill a process, '1' to show individual CPU cores.
kill
Sends signals to processes
$ kill 1234 # Send SIGTERM (graceful)
$ kill -9 1234 # Send SIGKILL (forceful)
$ kill -15 1234 # Send SIGTERM explicitly
$ killall firefox # Kill all firefox processes
jobs / bg / fg
Manage background jobs
$ sleep 100 & # Run in background
[1] 1234
$ jobs # List background jobs
[1]+ Running sleep 100 &
$ fg %1 # Bring job 1 to foreground
$ bg %1 # Send job 1 to background
nice / renice
Control process priority
$ nice -n 10 long_process.sh # Start with lower priority
$ renice -n 5 -p 1234 # Change priority of running process
Nice values range from -20 (highest priority) to 19 (lowest priority).
shutdown
System shutdown commands
$ sudo shutdown -h now # Shutdown immediately
$ sudo shutdown -h +10 # Shutdown in 10 minutes
$ sudo shutdown -r now # Reboot now
$ sudo shutdown -c # Cancel shutdown

Text Processing & Redirection

Redirection Operators
Redirect input/output streams
$ echo "Hello World" > file.txt # Overwrite file
$ echo "New line" >> file.txt # Append to file
$ cat < input.txt # Input from file
$ command 2> error.log # Redirect errors
$ command > output.txt 2>&1 # Redirect both output and errors
Pipe |
Connect command output to input
$ ls -l | grep "txt"
-rw-r--r-- 1 user user 1024 Aug 4 21:40 document.txt
-rw-r--r-- 1 user user 512 Aug 4 20:30 notes.txt
$ ps aux | grep apache | wc -l
tee
Write to both file and stdout
$ ls -l | tee directory_listing.txt
$ echo "Log entry" | tee -a logfile.txt
Use -a flag to append instead of overwriting the file.
grep
Search for patterns in text
$ grep "error" /var/log/syslog
$ grep -i "warning" logfile.txt # Case insensitive
$ grep -v "debug" app.log # Exclude lines with "debug"
$ grep -r "TODO" /home/user/code/ # Recursive search
cut
Extract columns from text
$ cut -d: -f1 /etc/passwd # Extract usernames
root
daemon
bin
sys
$ cut -c1-5 file.txt # Extract characters 1-5
$ echo "one,two,three" | cut -d, -f2
two
sort
Sort lines of text
$ sort names.txt
$ sort -n numbers.txt # Numerical sort
$ sort -r file.txt # Reverse sort
$ sort -k2 -t: /etc/passwd # Sort by 2nd field, colon delimiter
wc
Count lines, words, characters
$ wc file.txt
45 200 1024 file.txt
$ wc -l /etc/passwd # Count lines only
$ wc -w essay.txt # Count words only
Output format: lines words characters filename
tr
Translate or delete characters
$ echo "hello" | tr 'a-z' 'A-Z'
HELLO
$ echo "hello world" | tr -d ' '
helloworld
$ echo "hello world" | tr -s ' '
hello world

Archiving & Compression

tar
Archive files and directories
$ tar -cvf backup.tar /home/user/docs/
$ tar -xvf backup.tar # Extract archive
$ tar -tvf backup.tar # List archive contents
$ tar -czf backup.tar.gz /home/user/ # Create compressed archive
$ tar -xzf backup.tar.gz # Extract compressed archive
c=create, x=extract, t=list, v=verbose, f=file, z=gzip, j=bzip2
gzip / gunzip
Compress/decompress files with gzip
$ gzip largefile.txt
Creates: largefile.txt.gz
$ gunzip largefile.txt.gz
$ gzip -d largefile.txt.gz # Alternative to gunzip
bzip2 / bunzip2
Compress/decompress with better compression
$ bzip2 largefile.txt
Creates: largefile.txt.bz2
$ bunzip2 largefile.txt.bz2

Permissions & Special Bits

Set UID
Execute file as owner (4000)
$ chmod 4755 program
$ ls -l program
-rwsr-xr-x 1 root root 12345 Aug 4 21:40 program
Notice the 's' in owner execute position. Allows users to run program with root privileges.
Set GID
Files inherit directory group (2000)
$ chmod 2755 shared_directory
$ ls -ld shared_directory
drwxr-sr-x 2 user developers 4096 Aug 4 21:40 shared_directory
Sticky Bit
Only owner can delete files (1000)
$ chmod 1755 /tmp
$ ls -ld /tmp
drwxr-xr-t 10 root root 4096 Aug 4 21:40 /tmp
Notice the 't' in others execute position. Prevents users from deleting others' files.
umask
Set default permissions for new files
$ umask
0022
$ umask 027 # New files: 640, New dirs: 750
umask subtracts permissions from defaults (666 for files, 777 for directories).

Other Useful Commands

find
Search for files and directories
$ find /home -name "*.txt"
$ find /var -size +100M # Files larger than 100MB
$ find /tmp -mtime -1 # Files modified in last day
$ find /home -user john -type f # Files owned by john
du
Display directory space usage
$ du -sh /home/user
2.1G /home/user
$ du -h --max-depth=1 /var/log
1.2M /var/log/apache2
856K /var/log/apt
2.4M /var/log
which / whereis
Locate commands and their files
$ which python3
/usr/bin/python3
$ whereis python3
python3: /usr/bin/python3 /usr/bin/python3.8 /usr/lib/python3 /etc/python3 /usr/share/man/man1/python3.1.gz
history
Display command history
$ history | tail -5
497 ls -la
498 cd Documents
499 cat file.txt
500 history
$ !498 # Execute command 498 from history
$ !! # Execute last command
diff
Compare files line by line
$ diff file1.txt file2.txt
2c2
< This is line 2 in file1
---
> This is line 2 in file2
$ diff -u file1.txt file2.txt # Unified format
tree
Display directory structure as tree
$ tree /home/user/projects
/home/user/projects
├── web
│ ├── css
│ │ └── style.css
│ ├── js
│ │ └── script.js
│ └── index.html
└── python
└── main.py
