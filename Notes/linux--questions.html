Linux Commands & System Administration
Q1
To list your hidden files, use:
A) dot
B) ls -F
C) ls -a
D) ls -l

Correct Answer: C) ls -a

The ls -a command lists all files including hidden files (those starting with a dot). The -a flag stands for "all".

Q2
Which command is used to extract specific lines (records) from a file:
A) cut
B) cmp
C) grep
D) None of the above

Correct Answer: C) grep

The grep command searches for specific patterns and extracts matching lines from files. cut extracts columns, not lines.

Q3
What is the result of using the following command?
mkdir -p dir1/dir2/dir3
A) It creates three directories, named dir1, dir2, and dir1/dir2/dir3 respectively.
B) It creates three directories, named dir1, dir2, and dir3 in the current working directory.
C) It attempts to create dir1/dir2/dir3, it posts appropriate error messages.
D) It creates three directories, named dir1, dir2, dir3 in the root directory.

Correct Answer: A) It creates three directories, named dir1, dir2, and dir1/dir2/dir3 respectively

The -p flag creates parent directories as needed. It creates dir1, then dir1/dir2, then dir1/dir2/dir3 in a nested structure.

Q4
To list the processes running in the bash shell background:
A) ps
B) fg
C) bg
D) jobs

Correct Answer: D) jobs

The jobs command lists all active jobs in the current shell session, including background processes.

Q5
A user typed the following on his prompt:
mkdir dir1
touch dir1/file1
what will happen?
A) Error message will be displayed and nothing created
B) The user will be able to create the file only
C) The user will be able to create the directory only
D) The user will be able to create the directory and file

Correct Answer: D) The user will be able to create the directory and file

First mkdir dir1 creates the directory, then touch dir1/file1 creates the file inside that directory. Both commands will execute successfully.

Q6
What is the minimum permission needed to delete a file
A) write
B) read
C) read and write
D) No permissions

Correct Answer: A) write

To delete a file, you need write permission on the directory containing the file, not on the file itself.

Q8
What command displays your current username?
A) whoami
B) who
C) user
D) login

Correct Answer: A) whoami

The whoami command displays the current username. who shows all logged in users.

Q9
The LINUX command to print the system time is:
A) time
B) date
C) echo date
D) echo time

Correct Answer: B) date

The date command displays the current system date and time. time is used to measure execution time of commands.

Q10
To view inode number of a file?
A) ls -l filename
B) ls -i filename
C) ls -k filename
D) ls -a filename

Correct Answer: B) ls -i filename

The ls -i command displays the inode number of files. The -i flag stands for "inode".

Q12
How do you determine a PID?
A) The ps command is the only command for determining a PID.
B) The pgrep command is the only command for determining a PID.
C) Both the ps and pgrep commands display PIDs.
D) None of the above

Correct Answer: C) Both the ps and pgrep commands display PIDs

Both ps and pgrep can be used to find Process IDs. ps lists running processes with their PIDs, while pgrep finds PIDs by process name.

Q13
The output of the command ls -a is:
. ..tst dir1 file1
which command copies file1 into the dir1 directory?
A) cp /file1 dir1
B) cp -r dir1
C) cp file1 dir1
D) cp dir1 file1

Correct Answer: C) cp file1 dir1

The cp file1 dir1 command copies file1 into the dir1 directory. The syntax is cp source destination.

Q14
The number of data items stored in /etc/passwd for each user is:
A) 3
B) 5
C) 7
D) 9

Correct Answer: C) 7

Each line in /etc/passwd contains 7 fields separated by colons: username, password, UID, GID, GECOS, home directory, and shell.

Q15
What command shows you CPU and memory utilization for running processes?
A) top
B) tar
C) gzip
D) zip

Correct Answer: A) top

The top command displays real-time information about running processes, including CPU and memory usage.

Q16
To list the values of the Environment variables:
A) echo $ENV
B) ENV
C) env
D) None of the above

Correct Answer: C) env

The env command displays all environment variables. You can also use printenv for the same purpose.

Q17
What command allows you to add new secondary group?
A) useradd
B) usermod
C) groupmod
D) groupmems

Correct Answer: B) usermod

The usermod -aG groupname username command adds a user to a secondary group. The -a flag appends to existing groups.

Q18
Which statement display the lines from 3 to 9 from a file:
A) head -9 filename | tail -6
B) tail -3 filename | head -6
C) tail +3 filename | head -7
D) head -9 filename | tail -3

Correct Answer: A) head -9 filename | tail -6

head -9 gets the first 9 lines, then tail -6 gets the last 6 lines of those 9, which gives lines 4-9. For lines 3-9, we need 7 lines total.

Q19
What command allows you to open and view a file one page at a time?
A) more
B) cat
C) cp
D) view

Correct Answer: A) more

The more command displays file content one page at a time. less is a more advanced alternative with more features.

Q20
What are the minimum permission needed for running command ls -l:
A) read
B) execute
C) write and execute
D) read and execute

Correct Answer: D) read and execute

To run ls -l on a directory, you need read permission to list the contents and execute permission to access the directory.

Q21
State the initialization files of bash
A) .bashrc
B) /etc/bashrc
C) .bash_profile
D) /etc/profile

Correct Answer: All of the above

Bash initialization files include: .bashrc (user-specific), /etc/bashrc (system-wide), .bash_profile (login shells), and /etc/profile (system-wide login).

Q23
Which command is used to change default permissions for files and directories at the time of creation?
A) chmod
B) chown
C) useradd
D) umask

Correct Answer: D) umask

The umask command sets the default permissions for newly created files and directories by specifying which permissions to mask out.

Q24
Which tar option enables you to add one single file to a tar archive?
A) -a
B) -A
C) -r
D) -u

Correct Answer: C) -r

The tar -r option appends files to an existing tar archive. The -u option updates files that are newer.

Q25
Which key do you press to change into command mode while using vi?
A) CONTROL
B) ENTER
C) COMMAND
D) ESC

Correct Answer: D) ESC

In vi editor, pressing ESC switches from insert mode to command mode, allowing you to execute commands.

Q26
Which of the following commands would only show lines in the file /etc/passwd that start with the text anna?
A) grep anna /etc/passwd
B) grep -v anna /etc/passwd
C) grep $anna /etc/passwd
D) grep ^anna /etc/passwd

Correct Answer: D) grep ^anna /etc/passwd

The ^ symbol is a regex anchor that matches the beginning of a line. So ^anna matches lines that start with "anna".

Q27
Which key combination enables you to cancel a current interactive shell job?
A) ctrl-C
B) ctrl-D
C) ctrl-Z
D) ctrl-Break

Correct Answer: A) ctrl-C

Ctrl-C sends SIGINT to terminate the current process. Ctrl-Z suspends the process, and Ctrl-D sends EOF.

Q29
Which one of the following will output an error messages? (Note: you have read access on file1)
A) more file1
B) more file1 | cat
C) cat file1 | more
D) None of the above

Correct Answer: D) None of the above

All commands will work correctly since you have read access to file1. All three commands will display the file content successfully.

Q30
Which command can be used to list all groups a user is member of?
A) userlist
B) grouplist
C) id
D) groups

Correct Answer: C) id and D) groups

Both id and groups commands can list all groups a user belongs to. id shows more detailed information including UID and GID.

Q31
In general the working directory of the user root is
A) /
B) /root
C) /admin
D) /home/root

Correct Answer: B) /root

The root user's home directory is typically /root, not /home/root like regular users.

Q32
What is the command used to run a program with elevated permissions?
A) sed
B) mkdir
C) sudo
D) vip

Correct Answer: C) sudo

The sudo command allows users to run programs with elevated privileges, typically as root.

Q34
Which of the following is a relative path: ____________.
A) etc/passwd
B) /etc/passwd
C) //etc/passwd
D) ~/etc/passwd

Correct Answer: A) etc/passwd

A relative path doesn't start with /. Paths starting with / are absolute paths that start from the root directory.

Q35
What does the following command will do?
$ chmod 888 file
A) It enables r,w,x permissions for all users
B) It disable r,w,x permissions for all users
C) It enables r,w,x permissions only for owner of the file
D) None of the above

Correct Answer: D) None of the above

888 is not a valid octal permission value. Valid octal digits are 0-7. This command would result in an error.

Q36
In Linux, CD stands for ____________.
A) Current Directory
B) Current Document
C) Change Directory
D) Change Document

Correct Answer: C) Change Directory

The cd command stands for "Change Directory" and is used to navigate between directories in the filesystem.

Q37
Command which is used for moving to root directory in Linux
A) cd .
B) cd -
C) cd ..
D) cd /

Correct Answer: D) cd /

cd / changes to the root directory. cd . stays in current directory, cd .. goes to parent directory, cd - goes to previous directory.

Q38
Which of the following is an standard output redirection:
A) >
B) 2>
C) &
D) |

Correct Answer: A) >

> redirects standard output. 2> redirects standard error, | is a pipe, and & runs commands in background.

Q39
Hidden Files in Linux begin with a ____________.
A) $ Symbol
B) . Symbol
C) * Symbol
D) # Symbol

Correct Answer: B) . Symbol

Hidden files in Linux start with a dot (.). Examples include .bashrc, .profile, .ssh, etc.

Q40
Command which is used to remove empty directory in Linux
A) rd
B) rmdir
C) rdir
D) None of this

Correct Answer: B) rmdir

The rmdir command removes empty directories. For non-empty directories, use rm -r.

Q41
The location of user accounts database is stored in
A) /usr/bin/passwd
B) /var/shadow
C) /var/passwd
D) /etc/passwd

Correct Answer: D) /etc/passwd

User account information is stored in /etc/passwd. Password hashes are stored in /etc/shadow for security.

Q42
When using vi, what happens if you type the following ESC dd
A) You delete the current line.
B) You move the current line after the next line
C) You move the current line to beginning of file.
D) Nothing will happen to the file

Correct Answer: A) You delete the current line

ESC enters command mode, and dd in vi deletes the current line.

Q43
The kill command is used to:
A) send a kill signal only to a process
B) send a signal to a process
C) kill a signal
D) none of the above

Correct Answer: B) send a signal to a process

The kill command sends various signals to processes, not just the kill signal. Default is SIGTERM (15), but it can send any signal.

Q44
Which of the following permissions on a directory allow any user to cd to it
A) dr--r--r--
B) dr-xr--r--
C) dr-xr-xr-x
D) None of the above

Correct Answer: C) dr-xr-xr-x

To cd into a directory, you need execute (x) permission. Option C gives execute permission to owner, group, and others.

Q45
Which command is used to extract specific columns (fields) from a file:
A) grep
B) cut
C) cmp
D) None of the above

Correct Answer: B) cut

The cut command extracts specific columns or fields from files. grep extracts lines, not columns.

Q46
To remove (delete) a non-empty directory:
A) rmdir dir1
B) deldir dir1
C) rm -r dir1
D) All of the above

Correct Answer: C) rm -r dir1

rm -r recursively removes directories and their contents. rmdir only works on empty directories.

Q47
The command wc myfile will
A) Shows the number of lines in myfile.
B) Shows the number of characters in myfile
C) Shows the number of words in myfile.
D) All the above.

Correct Answer: D) All the above

The wc command displays line count, word count, and character count by default.

Q48
What is the command used to display text files
A) pwd
B) ls
C) cat
D) man

Correct Answer: C) cat

The cat command displays the contents of text files. Other options: more, less, head, tail.

Q49
What is the command used to display manual pages
A) cat
B) ls
C) manual
D) man

Correct Answer: D) man

The man command displays manual pages for commands and system functions.

Q51
Which of the following are correct commands for changing the current directory to the user's home? (Choose TWO answers)
A) cd /home
B) cd -
C) cd ~
D) cd
E) cd /

Correct Answer: C) cd ~ and D) cd

Both cd (without arguments) and cd ~ change to the user's home directory.

Q52
Command which is used to rename directory in Linux.
A) rd
B) rdir
C) mv
D) mvd

Correct Answer: C) mv

The mv command is used to move/rename files and directories in Linux.

Q53
Linux is
A) Single-User Operating System
B) Multi-User Operating System
C) Double-User Operating System
D) None of the above

Correct Answer: B) Multi-User Operating System

Linux is a multi-user operating system that allows multiple users to access the system simultaneously.

Q54
Which permission gives you authority to modify the contents of a file?
A) Read
B) Write
C) Execute
D) All of the above

Correct Answer: B) Write

Write permission allows you to modify the contents of a file. Read allows viewing, execute allows running.

Q55
Root of file system in Linux is denoted by
A) /
B) $
C) ~
D) #

Correct Answer: A) /

The root directory of the Linux filesystem is represented by a forward slash (/).

Q56
Symbol which indicates that user in working in home directory in Linux.
A) ~
B) /
C) $
D) #

Correct Answer: A) ~

The tilde (~) symbol represents the user's home directory in Linux.

Q57
Command to see list of files in Linux.
A) if
B) ls
C) lsfi
D) dir

Correct Answer: B) ls

The ls command lists files and directories in the current or specified directory.

Q58
Select operator which is used to remove permission in symbolic mode
A) +
B) -
C) =
D) *

Correct Answer: B) -

In chmod symbolic mode, - removes permissions, + adds permissions, and = sets exact permissions.

Q59
Symbol which is used for pipe in Linux.
A) &
B) |
C) >
D) <

Correct Answer: B) |

The pipe symbol (|) connects the output of one command to the input of another command.

Q60
Following file contains the encrypted password of the users.
A) /etc/passwd
B) /etc/shadow
C) /etc/gshadow
D) /etc/user.conf

Correct Answer: B) /etc/shadow

The /etc/shadow file contains encrypted user passwords. /etc/passwd contains user account information but not passwords.

Q61
Select vi editing command to delete character at cursor
A) x
B) d
C) dd
D) del

Correct Answer: A) x

In vi command mode, x deletes the character at the cursor position. d requires a motion command, dd deletes the entire line.

Q62
Which command lists the names of modules that are currently loaded in the kernel?
A) modprobe
B) lsmod
C) insmod
D) make

Correct Answer: B) lsmod

The lsmod command lists all currently loaded kernel modules. modprobe loads modules, insmod inserts modules.

Q64
Which of the following is the first step of the boot process?
A) BIOS
B) MBR
C) GRUB
D) init

Correct Answer: A) BIOS

BIOS (Basic Input/Output System) is the first step in the boot process, followed by MBR, bootloader (GRUB), kernel, and init.

Q65
Mona is a user who has returned from vacation and needs to have her account reactivated. Which of the following commands can you use to activate the mona account?
A) usermod -U mona
B) passwd -u mona
C) unlock mona
D) useradd mona

Correct Answer: B) passwd -u mona

The passwd -u command unlocks a user account. usermod -U also unlocks accounts in some systems.

Q66
By default a file can be a member in more than one group
A) True
B) False

Correct Answer: B) False

A file can only belong to one primary group, though the owner can belong to multiple groups. Each file has one owner and one group.

Q68
What command should you use to modify the /etc/sudoers file?
A) vigr
B) vipw
C) vi /etc/sudoers
D) visudo

Correct Answer: D) visudo

The visudo command safely edits the /etc/sudoers file with syntax checking to prevent syntax errors that could lock you out.
