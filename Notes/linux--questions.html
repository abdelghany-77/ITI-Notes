Q1
What does this command sequence do?
cat < file1 >> file2 | file3
A. file1 content will be appended to file2 and finally stored in file3
B. file1 content will be appended to file2 and file3 will be ignored
C. file2 and file3 will have same content
D. syntax error

Correct Answer: D. syntax error

This command has invalid syntax. You cannot use both redirection (>>) and piping (|) in this manner. The shell cannot interpret this mixed syntax - you must choose either redirection OR piping, not both together in this way.

Q2
In which directory can you store system user default files used for creating user directories?
A. /usr/tmp
B. /etc/skel
C. /etc/default
D. None of the above

Correct Answer: B. /etc/skel

The /etc/skel directory contains template files and directories that are automatically copied to a new user's home directory when the user account is created with useradd. This provides default files like .bashrc, .profile, etc.

Q3
Display start from line 6 and show next 15 lines of a large log file.
A. head -n 20 logfile | tail -n 15
B. head -n 15 logfile | tail -n 5
C. head -n 15 logfile | tail -n 6
D. head -n 6 logfile | tail -n 15

Correct Answer: A. head -n 20 logfile | tail -n 15

To display lines 6-20 (15 lines starting from line 6): First get the first 20 lines with head, then get the last 15 of those with tail. This gives you lines 6 through 20 (head - tail +1) -> 20 - 6 + 1= 15 then head -n 20 logfile | tail -n 15.

Q16

The nice command starts a process with a modified scheduling priority. Lower nice values (-20 to 0) mean higher priority, higher values (0 to 19) mean lower priority.

Q17

Signal 9 (SIGKILL) forcefully terminates a process and cannot be caught or ignored by the process. Signal 15 (SIGTERM) is the default graceful termination signal that processes can handle.
