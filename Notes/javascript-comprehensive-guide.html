Day 1: Introduction to JavaScript & Basic Concepts

1. What is JavaScript?

JavaScript Definition

JavaScript is a simple, flexible, lightweight, and interpreted scripting language. It's most well-known as the scripting language for web pages, used for interactivity with users, dynamic updates, and creating dynamic websites.

Key Characteristics

  • Client-side: Runs in the web browser
  • Object-based: Prototype-based, not class-based like Java
  • Interpreted: No compilation needed
  • Dynamic: Can modify content in real-time
  • Lightweight: Minimal overhead
Your First JavaScript Program
// Simple console output
console.log("Hello, World!");
// Interactive user input
alert("Welcome to JavaScript!");
let userName = prompt("What's your name?");
document.write("Hello, " + userName + "!");

4. JavaScript History & Evolution

Timeline of JavaScript

  • 1995: Created by Brendan Eich at Netscape as "LiveScript"
  • 1995: Renamed to "JavaScript" (marketing deal with Sun)
  • 1996: Submitted to ECMA International for standardization
  • 1997: ECMAScript standard established
  • 2000s: Browser wars, multiple implementations
  • 2009: ECMAScript 5 (ES5) released
  • 2015: ECMAScript 6 (ES6/ES2015) major update
  • 2015+: Yearly release cycles (ES2016, ES2017, etc.)

1. What is AJAX?

AJAX Definition

AJAX (Asynchronous JavaScript and XML) is a technique for creating interactive web applications. It allows web pages to update content dynamically without reloading the entire page.

Key Benefits: Better user experience, reduced server load, faster page interactions, real-time data updates

AJAX Characteristics

  • Asynchronous: Doesn't block the user interface
  • Background requests: Data exchange happens behind the scenes
  • Partial updates: Only specific parts of the page change
  • Multiple formats: Can work with XML, JSON, HTML, plain text
  • Real-time: Enables live data updates without page refresh
Traditional vs AJAX Approach
Traditional web approach:
// 1. User clicks a link or submits form

// 4. Server sends back complete HTML page
// 5. Browser reloads entire page
// Page flickers, slow user experience
AJAX approach:
// 1. JavaScript makes background request
// 2. Server sends back only data (JSON/XML)
// 3. JavaScript updates specific page elements
// Smooth user experience, no page reload
Real-world AJAX examples:
// Google Search suggestions (autocomplete)

class BreakpointManager {
constructor() {

sm: 576,
md: 768,
lg: 992,
xl: 1200,
xxl: 1400
};
this.listeners = new Map();
this.currentBreakpoint = this.getCurrentBreakpoint();
window.addEventListener('resize', () => {
this.checkBreakpoint();
});
}
getCurrentBreakpoint() {
let width = window.innerWidth;
let current = 'xs';
for (let [name, minWidth] of Object.entries(this.breakpoints)) {
if (width >= minWidth) {
current = name;
}
}
return current;
}
checkBreakpoint() {
let newBreakpoint = this.getCurrentBreakpoint();
if (newBreakpoint !== this.currentBreakpoint) {
let oldBreakpoint = this.currentBreakpoint;
this.currentBreakpoint = newBreakpoint;
// Notify listeners
for (let [id, callback] of this.listeners) {
callback(newBreakpoint, oldBreakpoint);
}
}
}
onBreakpointChange(id, callback) {
this.listeners.set(id, callback);
}
removeListener(id) {
this.listeners.delete(id);
}
}

Drag and drop implementation:
class DragDrop {
constructor(element, options = {}) {
this.element = element;
this.isDragging = false;
this.startX = 0;
this.startY = 0;
this.offsetX = 0;
this.offsetY = 0;
this.options = {
constrainToParent: options.constrainToParent || false,
onDragStart: options.onDragStart || null,
onDrag: options.onDrag || null,
onDragEnd: options.onDragEnd || null
};
this.init();
}
init() {
this.element.style.position = 'absolute';
this.element.style.cursor = 'move';
this.element.addEventListener('mousedown', this.handleMouseDown.bind(this));
document.addEventListener('mousemove', this.handleMouseMove.bind(this));
document.addEventListener('mouseup', this.handleMouseUp.bind(this));
}
handleMouseDown(e) {
this.isDragging = true;
let rect = this.element.getBoundingClientRect();
this.offsetX = e.clientX - rect.left;
this.offsetY = e.clientY - rect.top;
this.element.style.zIndex = '1000';
if (this.options.onDragStart) {
this.options.onDragStart(e, this.element);
}
e.preventDefault();
}
handleMouseMove(e) {
if (!this.isDragging) return;
let newX = e.clientX - this.offsetX;
let newY = e.clientY - this.offsetY;
if (this.options.constrainToParent && this.element.parentElement) {
let parent = this.element.parentElement;
let parentRect = parent.getBoundingClientRect();
let elementRect = this.element.getBoundingClientRect();
newX = Math.max(0, Math.min(parentRect.width - elementRect.width, newX));
newY = Math.max(0, Math.min(parentRect.height - elementRect.height, newY));
}
setPosition(this.element, newX, newY);
if (this.options.onDrag) {
this.options.onDrag(e, this.element, newX, newY);
}
}
handleMouseUp(e) {
if (!this.isDragging) return;
this.isDragging = false;
this.element.style.zIndex = '';
if (this.options.onDragEnd) {
this.options.onDragEnd(e, this.element);
}
}
}
